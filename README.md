# typescript
[Alpha] Typescript client for kubernetes

Needs lots of work...

# Installing
Install nodejs

```sh
# Typescript modules
sudo npm install -g typescript ts-node

# Required libraries for the client
sudo npm install -g request bluebird js-yaml base-64
```

# Running
## With `ts-node`
```sh
ts-node example.ts
```

## With `tsc`
```sh
tsc example.ts
node example.js
```
